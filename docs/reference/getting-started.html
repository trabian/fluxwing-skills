Getting Started

Installation

Clone the Fluxwing skills repository:

git clone https://github.com/trabian/fluxwing-skills.git
cd fluxwing-skills

Install the skills to Claude Code:

./scripts/install.sh

Quick Start

  1. Open Claude Code
  2. Say: "Create a primary button component"
  3. The uxscii-component-creator skill activates automatically
  4. Component files are saved to ./fluxwing/components/
